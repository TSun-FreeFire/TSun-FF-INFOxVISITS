Free Fire API Integration

A powerful, unified API for fetching comprehensive player data and managing visit interactions.

GET /info/{server}/{uid}

Fetches detailed player profile information including basic stats, guild details, pets, and social info.

GET /visit/{server}/{uid}

Triggers a 5x batch process to send visits (likes) to the specified player. Process runs in the background.
